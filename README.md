# Banking-DataAnalysis
## 📌 Project Overview
This project analyzes a bank marketing dataset using Python to understand customer behavior and perform exploratory data analysis (EDA).

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Project Tasks
- Loaded and inspected the dataset
- Checked missing values and data types
- Generated summary statistics
- Created visualizations
- Built a correlation matrix
- Performed exploratory data analysis (EDA)

- bankmarketing.csv
